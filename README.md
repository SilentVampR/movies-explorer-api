# movies-explorer-api
Diploma API

Серверная часть дипломной работы

Маршруты пользователя:
- /signup (POST)
- /signin (POST)
- /signout (GET)
- /users/me (PATCH, GET)

Маршруты фильмов:
- /movies (GET, POST)
- /movies/:id (DELETE)

**Стек**

HTML, CSS, JS, React, MongoDB, NodeJS, API, nginx
